## 数据库部分
表单定义:
```
create table reactsql_user(
	userid int(11) primary key,
    psw varchar(36) default null,
    usertel int(11)
)ENGINE=InnoDB DEFAULT CHARSET=utf8;
```
第一张表就是用户数据
```
create table titles(
	titleid int(11) primary key,
    userid int(11),
    sbjct varchar(20),
    qstn varchar(180),
    frqcy int(8),
    series int(10)
)ENGINE=InnoDB DEFAULT CHARSET=utf8;
```
第二章表格用于记录题目
```
create table scores(
	userid int(11) primary key,
    highest int(4),
    titleid int(11)
)ENGINE=InnoDB DEFAULT CHARSET=utf8;
```
第三张表格用于记录用户的最高分，以及曾经错过的题目
